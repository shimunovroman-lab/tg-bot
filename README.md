# Проект Чат бот с информацией о погоде
### ________________________________________________________________
## Описание
#### 🌤 Погода за секунду

Узнайте актуальную погоду в любой точке мира прямо в Telegram. Просто отправьте название города или геолокацию — и бот покажет температуру, влажность, скорость ветра и прогноз на ближайшие дни. Без лишней рекламы и сложных меню.
#### _________________________________________________________________________________
## Используемые технологии
#### Проект разработан на языке Python с использованием следующего набора инструментов :Python - основной язык программирования

  1. pyTelegramBotAPI (telebot) - взаимодействие с Telegram Bot API, обработка сообщений

  2. requests - отправка HTTP-запросов к внешнему API погоды

  3. json - парсинг и обработка данных, полученных от API

  4. random - генерация вариативных ответов и советов пользователю 

  5. OpenWeatherMap API - источник актуальных метеорологических данных

## Как запустить данный проект
Для запуска данного проекта, необходимо:

1. Клонируйте репозиторий или скачайте файлы
2. Установите необходимые библиотеки через pip
3. Найдите переменные TOKEN и API_KEY и вставьте туда свой токен и ключ
4. Запустите скрипт
5. Уже в Telegram перейдите в бота, воспользуйтесь командой /start
